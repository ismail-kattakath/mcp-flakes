1. **Monorepo detection is critical** - 27% of servers use this pattern
2. **Published packages are fastest** - 6x faster builds than source
3. **Python needs multi-stage** - uv + proper entrypoints
4. **Smoke tests catch everything** - 100% issue detection
5. **Patterns are finite** - 10 patterns cover 95% of servers
6. **Attribution is automatable** - Agents can do it correctly
7. **Agent parallelism scales** - More agents = more throughput
8. **Documentation generates itself** - From manifests to README
