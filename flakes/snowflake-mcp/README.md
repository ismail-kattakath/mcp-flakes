# Snowflake MCP Server

**[DEPRECATED]** This community MCP server is no longer maintained. Please use the official [Snowflake MCP Server](https://docs.snowflake.com/en/user-guide/snowflake-cortex/cortex-agents-mcp) instead.

## Overview

Legacy MCP server for Snowflake with Cortex AI, object management, and SQL orchestration capabilities. This implementation is deprecated in favor of the official Snowflake-maintained server.

## Features (Legacy)

- **Cortex Search**: Query unstructured data for RAG applications
- **Cortex Analyst**: Query structured data with semantic modeling
- **Cortex Agent**: Orchestrate across data retrieval
- **Object Management**: Create, drop, and update Snowflake objects
- **SQL Execution**: Run SQL with permission controls
- **Semantic Views**: Discover and query semantic views

## Migration Notice

For all new deployments, use the official Snowflake MCP Server available at:
https://docs.snowflake.com/en/user-guide/snowflake-cortex/cortex-agents-mcp

## Configuration

Connection requires Snowflake credentials:
- Account identifier
- Username and password
- Warehouse, database, schema, and role
